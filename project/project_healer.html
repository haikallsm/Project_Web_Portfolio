Healer Medical App

Developed with Python (Kivy + KivyMD)

Healer Medical App adalah aplikasi manajemen rumah sakit berbasis Python yang dirancang menggunakan framework Kivy dan KivyMD. Aplikasi ini memiliki fitur login multi-role (pasien & pegawai), pendaftaran poli, konsultasi online, hingga sistem antrean digital.

  • Modul login dan registrasi dengan autentikasi berbasis database
  • Fitur pengelolaan data pasien, dokter, dan jadwal poli
  • Sistem antrean real-time menggunakan queue
  • Antarmuka modern dengan Material Design
