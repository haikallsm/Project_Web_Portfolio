Ground Control Station Web

Developed with html, css, Javascript, python

Ground Control Station Web adalah Website control sebuah drone yang dirancang menggunakan html, css, vanilla Javascript, dan python. Website ini menampilkan data drone secara realtime saat terbang bahkan bisa membuat misi.

  • Navigasi Map dan Setting Way point
  • Live Streaming saat mengudara
  • Data Update secara Real-time
