E-commerce via CLI

Developed with C++

E-commerce via CLI adalah program sistem belanja sederhana berbasis Command Line Interface (CLI) yang dikembangkan menggunakan bahasa C++. Program ini memungkinkan pengguna untuk melakukan login, menambah barang ke keranjang, melakukan checkout, serta melihat nota pembelian. Sistem ini juga mengimplementasikan konsep antrian (queue) untuk mengatur proses checkout secara efisien.

  • Fitur login dan registrasi pengguna berbasis data linkedlist
  • Menambahkan, menghapus, dan menampilkan daftar barang belanja
  • Proses checkout otomatis dengan perhitungan total dan nota pembelian
  • Menampilkan riwayat transaksi dan detail pesanan
  • Sistem antrian checkout menggunakan struktur data queue
